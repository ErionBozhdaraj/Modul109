Infos zu AWS und Swisscloud

Infos zu AWS:

  • Gründungsjahr: 2006
  • Anzahl Datacenter: AWS hat weltweit über 200 Datacenter, die in sogenannten Availability Zones (Verfügbarkeitszonen) zusammengefasst sind.
  • Regionen: AWS hat 32 geografische Regionen weltweit (Stand: Mai 2024).
  • In der Schweiz ein Datacenter?: Ja, AWS hat seit 2022 eine Region in der Schweiz (Zürich), die mehrere Availability Zones umfasst.
  • Umsatz: AWS erzielte 2023 einen Umsatz von rund 90 Milliarden US-Dollar.
  • Kunden: AWS hat Millionen von aktiven Kunden weltweit, darunter Start-ups, Grossunternehmen und öffentliche Organisationen.
  • Mitarbeiter: AWS beschäftigt weltweit über 70.000 Mitarbeiter.
  • Dienstmodelle: AWS bietet drei Hauptdienstmodelle an:
    • Infrastructure as a Service (IaaS)
    • Platform as a Service (PaaS)
    • Software as a Service (SaaS)
  • Kosten für eine EC2-Instanz: 20.78 USD Monatlich

Infos zu Swisscloud:

  • Gründungsjahr: 2015
  • Anzahl Datacenter: Swiss Cloud betreibt 4 Datacenter in der Schweiz.
  • Regionen: Swiss Cloud operiert ausschliesslich in der Schweiz
  • Umsatz: Detaillierte Umsatzzahlen für Swiss Cloud sind nicht öffentlich verfügbar. Swiss Cloud ist ein Teil des breiteren Schweizer IT-Dienstleistungsmarktes.
  • Kunden: Swiss Cloud hat zahlreiche Geschäftskunden in der Schweiz, einschliesslich kleiner und mittlerer Unternehmen (KMU) sowie Grossunternehmen.
  • Mitarbeiter: Swiss Cloud beschäftigt mehrere Dutzend Mitarbeiter, spezifische Zahlen sind nicht öffentlich zugänglich.
  • Dienstmodelle: Swiss Cloud bietet verschiedene Cloud-Dienstmodelle an:
    • Infrastructure as a Service (IaaS)
    • Platform as a Service (PaaS)
    • Software as a Service (SaaS)
    • Managed Services
